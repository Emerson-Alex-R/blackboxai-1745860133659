# Guinchos Nêne - Serviço de Auto Socorro 24 Horas

## Project Overview
Guinchos Nêne is a responsive web application designed to provide 24-hour roadside assistance services. This application serves as an online presence for the Guinchos Nêne company, offering functionalities such as service descriptions, a gallery, contact information, and an interactive map.

## Installation
To set up this project locally, follow the steps below:

1. **Clone the repository:**
   ```bash
   git clone https://github.com/YOUR_USERNAME/guinchos-nene.git
   ```

2. **Navigate into the project directory:**
   ```bash
   cd guinchos-nene
   ```

3. **Open the `index.html` file in a web browser:**
   Simply open `index.html` in your preferred web browser to view the application.

## Usage
Once the application is opened in a web browser, users can navigate through different sections:

- **Início:** Introduction and a call to action for contacting the service.
- **Quem Somos:** Information about the company and its services.
- **Serviços:** Display of service features and benefits.
- **Galeria:** A gallery showcasing images related to the services.
- **Contato:** Contact form and information for reaching out to the company.

## Features
- **Responsive Design:** Adapts to different screen sizes for optimal usability.
- **Service Descriptions:** Clear details on the services offered, including 24-hour assistance.
- **Image Gallery:** Displays images for visual representation of services.
- **Contact Form:** Allows users to reach out directly via email.
- **Interactive Map:** Provides location details using Google Maps integration.
- **Smooth Navigation:** Smooth scroll and transitions for a better user experience.

## Dependencies
This project uses the following libraries:
- **Tailwind CSS:** For styling the components.
- **Font Awesome:** For vector icons.

### Tailwind CSS
Included via CDN. The configuration is set within the `<script>` tag in the HTML.

### Font Awesome
Included via CDN for better visual representation of icons.
